Vision-based Hand Gesture Interface for Real-time Computer Operation Control

Can operate Computer with hand Gestures

In this project a user can control his PC using simple and friendly hand gestures.

Users can use point on the screen using the index finger and many actions like scroll, left click, right click, double click, etc.. can be done using the gestures.

Done using the simple camera with every PC. Software involved is mediapipe, DL, pyautogui.

We presented this project at the TENCON core C conference 2022(paper id 1844).
